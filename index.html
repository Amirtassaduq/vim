VIM CHEAT SHEET

GLOBAL

:h keyword
open help for keyword
:sav file
save file as
:clo
close current pane
:ter
open terminal window
K
open man page for word under cursor
Tip Run vimtutor in terminal to learn first Vim commands.

CURSOR MOVEMENT

h j k l
basic movement (left, down, up, right)
gj gk
move through wrapped lines
H M L
top/middle/bottom of screen
w W
start of next word (with/without punctuation)
e E
end of word (with/without punctuation)
b B
back to word start (with/without punctuation)
ge gE
end of previous word
%
jump to matching pair () {} []
0 ^ $ g_
line start/first non-blank/end/last non-blank
gg G
first/last line of document
5gg 5G
go to line 5
gd gD
local/global declaration
fx Fx
jump to next/previous occurrence of x
; ,
repeat last f/F/t/T movement forward/backward
} {
next/previous paragraph or code block
zz zt zb
center/top/bottom cursor on screen
Ctrl+e Ctrl+y
scroll down/up without moving cursor
Ctrl+b Ctrl+f
page up/down
Ctrl+d Ctrl+u
half page down/up

INSERT MODE

i I
insert at cursor/start of line
a A
append after cursor/end of line
o O
new line below/above
ea
append at end of word
Ctrl+h
delete previous character
Ctrl+w
delete previous word
Ctrl+j
insert line break
Ctrl+t
indent right
Ctrl+d
indent left
Ctrl+n
next autocomplete
Ctrl+p
previous autocomplete
Ctrl+r x
insert from register x
Ctrl+o x
temporary normal mode command x
Esc
exit insert mode
Ctrl+c
exit insert mode

EDITING

r
replace single character
R
replace multiple characters until ESC
J
join line below with space
gJ
join line below without space
gwip
reflow paragraph
g~
switch case up to motion
gu
change to lowercase up to motion
gU
change to uppercase up to motion
cc
change (replace) entire line
c$ C
change to end of line
ciw
change entire word
cw ce
change to end of word
s
delete char and substitute (same as cl)
S
delete line and substitute (same as cc)
xp
transpose two letters
u
undo
U
restore (undo) last changed line
Ctrl+r
redo
.
repeat last command
~
switch case of character
g~~
switch case of line
gUU guu
uppercase/lowercase line

MARKS

:marks
list of marks
ma
set current position for mark A
`a
jump to position of mark A
y`a
yank text to position of mark A
`0
go to last exit position
`"
go to last edit position
`.
go to last change position
``
go to position before last jump
:ju
list of jumps
:changes
list of changes
Tip Use backtick (`) to jump to exact mark position or apostrophe (') to jump to line start.

VISUAL MODE

v
start visual mode, mark lines
V
start linewise visual mode
Ctrl+v
start visual block mode
o
move to other end of marked area
O
move to other corner of block
aw
mark a word
ab
a block with ()
aB
a block with {}
at
a block with <> tags
ib
inner block with ()
iB
inner block with {}
it
inner block with <> tags
Esc Ctrl+c
exit visual mode
Tip Instead of b or B one can also use ( or { respectively.

VISUAL COMMANDS

>
shift text right
<
shift text left
y
yank (copy) marked text
d
delete marked text
~
switch case
u
change marked text to lowercase
U
change marked text to uppercase

REGISTERS

:reg
show registers content
"xy
yank into register x
"xp
paste contents of register x
"+y
yank into system clipboard
"+p
paste from system clipboard
Special registers:
0 - last yank
" - unnamed register
% - current file name
# - alternate file name
* - clipboard contents (X11 primary)
+ - clipboard contents (X11 clipboard)
/ - last search pattern
: - last command-line
. - last inserted text
- - last small delete
= - expression register
_ - black hole register

MACROS

qa
record macro a
q
stop recording macro
@a
run macro a
@@
rerun last run macro

CUT/PASTE

yy
yank (copy) a line
2yy
yank 2 lines
yw
yank from cursor to next word start
yiw
yank word under cursor
yaw
yank word under cursor and space
y$ Y
yank to end of line
p
paste after cursor
P
paste before cursor
gp
paste after and leave cursor after
gP
paste before and leave cursor after
dd
delete (cut) a line
2dd
delete 2 lines
dw
delete from cursor to next word start
diw
delete word under cursor
daw
delete word under cursor and space
:3,5d
delete lines 3 to 5
d$ D
delete to end of line
x
delete character
Range Examples:
:.,$d - Current line to EOF
:.,1d - Current line to file start
:10,1d - Line 10 to file start

EXITING

:w
write (save) file, don't exit
:w !sudo tee %
write file using sudo
:wq :x ZZ
write (save) and quit
:q
quit (fails if unsaved changes)
:q! ZQ
quit and discard changes
:wqa
write and quit all tabs
:qa!
quit all tabs and discard changes

SEARCH/REPLACE

/pattern
search forward for pattern
?pattern
search backward for pattern
\vpattern
'very magic' regex pattern (no escaping needed)
n
repeat search in same direction
N
repeat search in opposite direction
:%s/old/new/g
replace all old with new throughout file
:%s/old/new/gc
replace all with confirmations
:noh
remove search highlighting

SEARCH IN MULTIPLE FILES

:vim /pattern/ {file}
search for pattern in multiple files
Example: :vim /foo/ **/*
:cn
jump to next match
:cp
jump to previous match
:copen
open quickfix window with matches
:cclose
close quickfix window

TABS

:tabnew
open new tab
:tabnew {file}
open file in new tab
Ctrl+wT
move current window to new tab
gt :tabn
next tab
gT :tabp
previous tab
3gt
go to tab 3
:tabm 2
move tab to position 2
:tabc
close current tab
:tabo
close all other tabs

WORKING WITH MULTIPLE FILES

:e file
edit file in new buffer
:bn
next buffer
:bp
previous buffer
:bd
delete buffer (close file)
:b #
go to buffer by index #
:b file
go to buffer by filename
:ls :buffers
list all open buffers
:sp file
open file in new buffer (horizontal split)
:vs file
open file in new buffer (vertical split)
:vert ba
edit all buffers as vertical windows
:tab ba
edit all buffers as tabs
Ctrl+ws
split window
Ctrl+wv
split window vertically
Ctrl+ww
switch windows
Ctrl+wq
quit window
Ctrl+wx
exchange current window with next
Ctrl+w=
equalize window sizes
Ctrl+wh Ctrl+wl
move to left/right window (vertical split)
Ctrl+wj Ctrl+wk
move to window below/above (horizontal split)
Ctrl+wH Ctrl+wL Ctrl+wJ Ctrl+wK
move windows to far left/right/bottom/top

INDENT TEXT

>>
indent line right one shiftwidth
<<
de-indent line left one shiftwidth
>%
indent block with () or {} (cursor on brace)
<%
de-indent block with () or {} (cursor on brace)
>ib
indent inner block with ()
>at
indent block with <> tags
3==
re-indent 3 lines
=%
re-indent block with () or {} (cursor on brace)
gg=G
re-indent entire buffer

DIFF

zf
manually define fold up to motion
zd
delete fold under cursor
za
toggle fold under cursor
zo
open fold under cursor
zc
close fold under cursor
zr
reduce (open) all folds by one level
zm
fold more (close) all folds by one level
zi
toggle folding functionality
]c

[c
jump to start of previous change
do :diffg

dp :diffpu
